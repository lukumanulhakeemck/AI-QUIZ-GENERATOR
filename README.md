# AI-QUIZ-GENERATOR
To automate the creation of study quizzes from PDF textbooks
##Tools:
 - Streamlit
 - PyMuPDF
 - Google Generative AI
 - OS and re (Regex)
##Core Technology:
 - Powered by Google Gemini Pro (LLM) and Python
##Result:
 - The system reads the text and generates specific questions
 - User takes the quiz in a clean web interface
 - Get scores and detailed AI explanations (XAI) for every answer
